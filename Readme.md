## Babel

#### Build
```BASH
./RUN.sh
```

### Server TCP
Default port: **2000**

#### Run
```BASH
./RUN.sh server
```

#### Commands
```
/connect
/disconnect
/join
/leave
/call
```

### Client
Default port: **2000**

#### Run
```BASH
./RUN.sh client
```
